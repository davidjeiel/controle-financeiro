# Tratar erros recorrentes

Use este guia para diagnosticar falhas comuns em execucao, integracao ou publicacao.

## Procedimento

1. Identifique o codigo de erro retornado.
2. Consulte a referencia de codigos de erro.
3. Verifique variaveis de ambiente e credenciais.
4. Reproduza a chamada localmente.
5. Registre a causa raiz e a acao corretiva.

## Boas praticas

- Inclua identificadores de correlacao nos logs.
- Evite expor dados sensiveis nas mensagens.
- Transforme erros recorrentes em testes automatizados.
