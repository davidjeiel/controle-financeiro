# Publicar no gateway

Use este guia para publicar uma API no gateway corporativo.

## Procedimento

1. Confirme que o contrato da API esta revisado.
2. Configure rota, upstream e politica de autenticacao.
3. Defina limites de taxa e timeout.
4. Publique em ambiente de homologacao.
5. Execute testes de contrato e observabilidade.
6. Promova para producao apos aprovacao.

## Validacao

A API deve estar acessivel pelo gateway, com autenticacao ativa e logs rastreaveis.
