# Executando projeto localmente

Este tutorial mostra como executar a aplicacao em ambiente de desenvolvimento.

## Pre-requisitos

- Node.js instalado.
- Dependencias do projeto instaladas.
- Repositorio clonado localmente.

## Passos

1. Instale as dependencias:

```shell
npm install
```

2. Inicie o servidor de desenvolvimento:

```shell
npm start
```

3. Abra `http://localhost:3000` no navegador.

## Resultado esperado

A interface do controle financeiro deve carregar localmente para validacao funcional.
