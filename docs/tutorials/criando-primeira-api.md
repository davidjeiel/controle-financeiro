# Criando primeira API

Este tutorial orienta a criacao de uma primeira API para integracao com o sistema financeiro.

## Objetivo

Ao final, voce tera um endpoint simples preparado para receber dados financeiros e validar o fluxo de entrada da aplicacao.

## Passos

1. Defina o recurso que sera exposto pela API.
2. Crie o contrato inicial com metodo, rota e payload esperado.
3. Implemente uma resposta minima de sucesso.
4. Registre exemplos de requisicao e resposta.
5. Valide o comportamento localmente antes da publicacao.

## Resultado esperado

A API deve responder com status de sucesso e retornar um corpo JSON previsivel para uso em testes e integracoes.
