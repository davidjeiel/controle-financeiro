# Arquitetura

Esta explicacao descreve a organizacao tecnica do projeto.

## Visao geral

A aplicacao e um frontend React para controle financeiro pessoal. A interface permite registrar entradas e saidas, calcular saldo e exportar dados.

## Componentes principais

- `src/App.jsx`: composicao principal da aplicacao.
- `src/components/Form`: entrada de novas transacoes.
- `src/components/Grid`: listagem das transacoes.
- `src/components/Resume`: indicadores de receita, despesa e saldo.
- `src/styles`: estilos globais.

## Publicacao

O build React e publicado na raiz do GitHub Pages. A documentacao Diataxis e publicada em `/documentation/`.
