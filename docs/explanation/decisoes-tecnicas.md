# Decisoes tecnicas

Esta pagina registra decisoes relevantes para evolucao do projeto.

## React como camada de interface

React foi adotado para organizar a interface em componentes reutilizaveis e facilitar a evolucao de formularios, listagens e indicadores financeiros.

## Documentacao como codigo

A documentacao fica no repositorio em Markdown para permitir revisao por pull request, historico de alteracoes e publicacao automatica.

## GitHub Pages

GitHub Pages e usado para disponibilizar a demo e a documentacao sem infraestrutura adicional.
