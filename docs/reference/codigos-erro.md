# Codigos erro

Catalogo de codigos de erro esperados.

| Codigo | Cenario | Acao recomendada |
| --- | --- | --- |
| `AUTH_001` | Token ausente ou invalido. | Renovar credenciais e repetir a chamada. |
| `VAL_001` | Payload invalido. | Corrigir campos obrigatorios. |
| `FIN_001` | Transacao nao encontrada. | Validar identificador enviado. |
| `SYS_001` | Erro inesperado. | Consultar logs e correlacao da requisicao. |
